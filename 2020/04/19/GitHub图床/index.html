使用github搭建免费图床

写博客的时候传图是一个非常麻烦的事儿，网上免费的传图网站要不图容易丢失，要不有各种现实。我呢，比较抠，又不想花钱，so，发现了可以利用github+picgo搭建一个免费的图床。哈哈

一、首先我们先需要在github中设置一下：
流程：新建 public 仓库 -> 创建 token -> 复制 token 备用

二、PicGo 客户端配置

1、下载picgo客户端
下载地址：https://github.com/Molunerfinn/PicGo

2、配置picgo中的GitHub设置

仓库名 即你的仓库名

分支名 默认 master

Token 就是刚刚复制的那一串字符

存储路径 这个可以填也可以不填，填了的话图片就上传到 git 中 data 这个文件夹

域名格式 https://raw.githubusercontent.com/[username]/[仓库名]/master

或者可以用加速域名：https://cdn.jsdelivr.net/gh/caisichong/image@master
然后点确定就可以了。

三、gicgo的简单使用
1、将图片放入传图区域后上传，上传成功后点击相册

2、打开相册

Markdown位置可选择需要上传的格式
然后勾选需要的图片，点击批量复制，拷贝到md文件即可。
