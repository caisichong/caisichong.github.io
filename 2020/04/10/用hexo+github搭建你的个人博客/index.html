用github和hexo搭建你的个人博客

1.先在github上注册一个账号，这里就不多赘述了。
2.创建一个仓库(repository) 来存储我们的网站，点击首页任意位置出现的 New repository按钮创建仓库,


Respository name 中的username.github.io 的username 一定与前面的Owner 一致，记住你的username下面会用到。

3.安装git。官网下载安卓配置后即可，就不多赘述了。
4.安装Nodejs。官网下载安卓配置后即可，就不多赘述了。
5.安装Hexo
以上所有都安装完成之后再安装Hexo
$ npm install hexo-cli -g

所有必须工具已经安装完成，下面我们就可以生成博客，上传至我们的Github 仓库了。

接下来我们需要用Hexo初始化一个博客，然后更改一些配置，加上自己喜欢的主题，
写上第一篇文章，然后发布到自己的个人Github网站(username.github.io)。

一、创建博客
将下面的 username 替换成你喜欢的名字，执行成功后，会创建出一个名为 username.github.io 的文件夹。
$ hexo init username.github.io

更改配置
主题安装
为了使博客不太难看，我们需要安装一个主题，切换至刚刚生成的Hexo 目录，安装主题
$ cd username.github.io
$ git clone https://github.com/iissnan/hexo-theme-next themes/next
这是个随便选的主题，大家可以自己去https://hexo.io/themes/下载更多的主题。
基础配置：打开文件位置username.github.io/_config.yml修改几个键值对，下面把几个必须设置的列出来按需求修改，记得保存， 还有注意配置的键值之间一定要有空格。更多设置…
title: 蔡蔡的小破网站 //你博客的名字
author: caicai //你的名字
language: zh-Hans //语言 中文
theme: next //刚刚安装的主题名称
deploy:
type: git //使用Git 发布
repo: https://github.com/username/username.github.io.git // 刚创建的Github仓库

主题配置：
主题配置文件在username.github.io/themes/next/_config.yml中修改。可以看下这个
http://theme-next.iissnan.com/getting-started.html

写文章
所有基础框架都已经创建完成，接下来可以开始写你的第一篇博客了
在username.github.io/source/_posts下创建你的第一个博客吧，例如，创建一个名为hello.md的文件，用Markdown写哈，也可以用其他的工具编写，注意保存哈。

测试一下你的网站：

$ hexo s

测试服务启动，你可以在浏览器中输入https://localhost:4000 访问了。

安装hexo-deployer-git自动部署发布工具

$ npm install hexo-deployer-git –save

发布
测试没问题后，我们就生成静态网页文件发布至我们的Github pages 中。
$ hexo clean && hexo g && hexo d
如果这是你的第一次，终端会让你输入Github 的邮箱和密码，正确输入后，骚等一会儿，就会把你的网站上传至Github 了。以后在每次把博客写完后，执行一下这个命令就可以直接发布了，

效果可以参考本网站哈
